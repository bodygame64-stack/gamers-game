# gamers-game
## Description
A simple 2D game developed using Python and Pygame.  
The player receives a random character (Warrior, Mage, Rogue) and battles a set of NPCs.  
The game features an HP system, normal attacks, and special attacks unique to each character type.

## Controls
### In Menu:
- Type your name using the keyboard
- `1` to start the campaign
- `Backspace` to delete a character
- `Esc` to quit

### In Campaign:
- `1 / 2 / 3` to get a random character
- `Enter` to start the duel if keeping the current character

### During Duel:
- The duel runs automatically, player and NPC take turns attacking

### After Result:
- If you win:
- `C` to keep your character
- `R` to reroll
- If you lose:
- `Enter` to return to the menu

### When Campaign is Finished:
- `Enter` to return to the main menu

---

## Notes
- You can change the background by placing a `background.png` image inside the `assets` folder.
- If no background image is found, a default background color will be used.
- The game involves randomness in attack outcomes and special moves.
